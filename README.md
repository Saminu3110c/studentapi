## Use the following command to run
- Navigate to project folder:
    ```bash
        cd studentapi
    ```
- And run:

    ```bash
        ./mvnw spring-boot:run
    ```
    ```bash
        http://localhost:8080/h2-console
    ```
    ```bash
        http://localhost:8080/students
    ```
